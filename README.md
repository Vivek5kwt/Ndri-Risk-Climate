# Ndri Risk Climate App

This repository now includes a simple Flutter demo showing how to preview and edit answers before submission.

- `mobile_app/lib/main.dart` contains a basic form screen where you can enter answers and tap **Preview** to open the preview screen.
- `mobile_app/lib/screens/preview_answers_screen.dart` displays all answers with options to edit specific questions or submit all.

Run the Flutter app from the `mobile_app` directory with `flutter run` to try it out.
